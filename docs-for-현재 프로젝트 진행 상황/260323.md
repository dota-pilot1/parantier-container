## 현재 구현된 핵심 기능

### 1. 인증/인가 시스템 ✅
- **JWT 기반 인증**
  - 로그인/로그아웃
  - 회원가입
  - 토큰 자동 갱신
- **역할 기반 권한 관리 (RBAC)**
  - ROLE_USER: 일반 사용자
  - ROLE_ADMIN: 관리자
  - Spring Security 통합
- **세션 관리**
  - LocalStorage 기반 토큰 저장
  - 앱 로드 시 자동 로그인 상태 복원
  - TanStack Store로 전역 상태 관리

**관련 파일**:
- Backend: parantier-api/src/main/java/com/mapo/palantier/config/SecurityConfig.java
- Backend: parantier-api/src/main/java/com/mapo/palantier/user/presentation/AuthController.java
- Frontend: parantier-front/src/entities/user/model/authStore.ts
- Frontend: parantier-front/src/features/auth/login/LoginForm.tsx

### 2. 사용자 관리 ✅
- **CRUD 기능**
  - 사용자 목록 조회 (페이지네이션)
  - 사용자 추가
  - 권한 변경 (USER ↔ ADMIN)
  - 활성화/비활성화
  - 사용자 삭제 (단일/다중)
- **AG-Grid 기반 고급 테이블**
  - 정렬 (모든 컬럼)
  - 필터링 (텍스트, 숫자, 날짜)
  - 페이지네이션 (10, 20, 50, 100)
  - 체크박스 다중 선택
  - 중앙 정렬된 셀/헤더
- **권한별 액션 제어**
  - ADMIN만 권한 변경 가능
  - ADMIN만 사용자 삭제 가능

**관련 파일**:
- Backend: parantier-api/src/main/java/com/mapo/palantier/user/application/UserService.java

- Frontend: parantier-front/src/pages/admin/users/UsersPage.tsx

### 3. 메뉴 관리 시스템 ✅
- **계층형 트리 구조**
  - 무제한 depth 지원
  - 부모-자식 관계 관리
  - 재귀적 렌더링
- **메뉴 타입**
  - HEADER: 상단 헤더 메뉴

- **CRUD 기능**
  - 트리 뷰에서 메뉴 선택
  - 우측 패널에서 상세 편집
  - 컨텍스트 메뉴 (우클릭)
  - 인라인 하위 메뉴 추가
  - 삭제 시 하위 메뉴 개수 표시
- **검색 기능**
  - Enter 기반 명시적 검색
  - 매칭된 메뉴 하이라이트 (노란색)
  - 부모 노드 자동 확장
  - 검색 결과 없을 때 전체 접기
  - 검색어 초기화 버튼
- **트리 조작**
  - 전체 펼치기/접기
  - 개별 노드 확장/축소
  - 아이콘으로 타입 구분 (📁 HEADER, 📄 기타)
- **권한 기반 메뉴 표시**
  - requiredRole로 접근 제어
  - 헤더/사이드바에 동적 렌더링

## 기술 스택

### Backend
```
Framework:    Spring Boot 3.x
ORM:          MyBatis
Database:     MySQL 8.x
Security:     Spring Security + JWT
Build:        Gradle
Language:     Java 17+
```

### Frontend
```
Framework:    React 18 + TypeScript
Router:       TanStack Router
State:        TanStack Store (Zustand-like)
Query:        TanStack Query (React Query)
UI:           Tailwind CSS + shadcn/ui
Grid:         AG-Grid Community
Notification: Sonner (Toast)
Build:        Vite
```

### Architecture
```
Backend:      DDD (Domain-Driven Design)
              - application/ (Service)
              - domain/ (Entity, Repository Interface)
              - infrastructure/ (MyBatis Mapper Implementation)
              - presentation/ (Controller, DTO)

Frontend:     Feature-Sliced Design (FSD)
              - app/ (Routes, Providers)
              - pages/ (Page Components)
              - widgets/ (Header, Sidebar)
              - features/ (Business Logic)
              - entities/ (API, Store)
              - shared/ (UI Components)
```
