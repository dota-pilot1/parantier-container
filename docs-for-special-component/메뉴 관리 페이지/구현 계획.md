# 메뉴 관리 페이지 구현 계획

## 개요
- 듀얼 뷰 방식: 왼쪽 트리 뷰 + 오른쪽 상세 편집
- 라이브러리 없이 순수 React로 구현
- 확장/축소 기능으로 대량 메뉴 대응

## 1. 데이터 구조

### Menu 인터페이스
```typescript
interface Menu {
  id: number;
  name: string;
  path: string;
  type: 'HEADER' | 'SIDE';
  parentId: number | null;
  permission: string;
  orderNum: number;
  active: boolean;
}
```

## 2. 컴포넌트 구조

```
MenuManagementPage
├─ TreePanel (왼쪽)
│  ├─ SearchBar (검색)
│  ├─ TreeView
│  │  └─ TreeNode (재귀)
│  └─ AddButton
└─ DetailPanel (오른쪽)
   ├─ MenuEditForm
   └─ EmptyState
```

## 3. 핵심 기능

### 3.1 트리 뷰 (왼쪽 패널)
- 계층 구조 표시 (인덴트)
- 확장/축소 토글 (▶/▼)
- 메뉴 선택 (클릭)
- 아이콘 표시 (HEADER: 📁, SIDE: 📄)
- 비활성 상태 뱃지
- 검색 기능

### 3.2 상세 편집 (오른쪽 패널)
- 메뉴 정보 폼
  - 메뉴명 (text)
  - 타입 (select: HEADER/SIDE)
  - 경로 (text)
  - 권한 (select)
  - 순서 (number)
  - 활성 (checkbox)
- 저장 버튼
- 삭제 버튼
- 선택 없을 때 안내 메시지

### 3.3 확장/축소 전략 (대량 메뉴 대응)

#### ⭐ 기본 동작: 전체 접힌 상태
- 초기 렌더링: 최상위 메뉴(parentId === null)만 표시
- 사용자가 필요한 메뉴만 확장
- **성능**: 수백 개 메뉴도 초기 렌더링 빠름
- **UX**: 구조 파악 용이, 스크롤 최소화

## 4. 구현 단계

### Phase 1: 기본 구조 (필수)
1. 레이아웃: Grid 2-column
2. 트리 뷰: 재귀 컴포넌트
3. 확장/축소: 기본 접힌 상태
4. 메뉴 선택: 상태 관리
5. 상세 폼: CRUD

### Phase 2: 사용성 개선 (권장)
1. 검색 기능
2. 검색 결과 자동 확장
3. 전체 접기/펼치기 버튼
4. 키보드 네비게이션

## 5. 핵심 코드

### 트리 노드 렌더링
```typescript
const TreeNode = ({ menu, depth, allMenus, expandedIds, selectedId, onSelect, onToggle }) => {
  const children = allMenus.filter(m => m.parentId === menu.id);
  const hasChildren = children.length > 0;
  const isExpanded = expandedIds.has(menu.id);
  const isSelected = menu.id === selectedId;

  return (
    <div style={{ paddingLeft: `${depth * 20}px` }}>
      <div
        className={`node ${isSelected ? 'selected' : ''}`}
        onClick={() => onSelect(menu)}
      >
        {hasChildren && (
          <span onClick={(e) => { e.stopPropagation(); onToggle(menu.id); }}>
            {isExpanded ? '▼' : '▶'}
          </span>
        )}
        <span>{menu.type === 'HEADER' ? '📁' : '📄'}</span>
        <span>{menu.name}</span>
      </div>

      {hasChildren && isExpanded && children.map(child => (
        <TreeNode
          key={child.id}
          menu={child}
          depth={depth + 1}
          allMenus={allMenus}
          expandedIds={expandedIds}
          selectedId={selectedId}
          onSelect={onSelect}
          onToggle={onToggle}
        />
      ))}
    </div>
  );
};
```
