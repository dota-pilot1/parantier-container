# 체크박스 설정 관련 변경 사항

## 변경 날짜
2026-03-22

---

## 문제 상황

**증상**: 체크박스가 이중으로 표시됨 (각 행마다 2개씩)

**원인**: AG-Grid v32.2에서 `rowSelection` 사용 방식이 변경됨

---

## 해결 방법

### AS-IS (구 방식 - 문제 있음)

```typescript
// 1. 체크박스 전용 컬럼 정의
const columnDefs = [
  {
    headerName: '',
    checkboxSelection: true,           // ❌ deprecated
    headerCheckboxSelection: true,     // ❌ deprecated
    width: 50,
    pinned: 'left',
  },
  { field: 'id', headerName: 'ID' },
  { field: 'email', headerName: '이메일' },
]

// 2. rowSelection 설정
<AgGridReact
  rowSelection={{ mode: 'multiRow' }}  // ❌ deprecated
  columnDefs={columnDefs}
/>
```

**문제**:
- `checkboxSelection` 속성이 deprecated
- `rowSelection={{ mode: 'multiRow' }}` 방식이 deprecated
- 체크박스가 이중으로 표시됨

---

### TO-BE (신 방식 - 해결)

```typescript
// 1. 체크박스 전용 컬럼 정의 제거 (불필요)
const columnDefs = [
  { field: 'id', headerName: 'ID' },
  { field: 'email', headerName: '이메일' },
]

// 2. rowSelection 문자열로 변경
<AgGridReact
  rowSelection="multiple"              // ✅ 새 방식
  columnDefs={columnDefs}
/>
```

**핵심 변경**:
- 체크박스 컬럼 정의 **삭제**
- `rowSelection="multiple"` 사용
- AG-Grid가 자동으로 체크박스 컬럼 추가

---

## 코드 변경 내용

### 변경 1: columnDefs에서 체크박스 컬럼 제거

```typescript
// BEFORE
const columnDefs = useMemo<ColDef<UserResponse>[]>(
  () => [
    {
      headerName: '',
      checkboxSelection: true,
      headerCheckboxSelection: true,
      width: 50,
      maxWidth: 50,
      pinned: 'left',
      suppressHeaderMenuButton: true,
      lockPosition: true,
      resizable: false,
      sortable: false,
      filter: false,
    },
    { field: 'id', headerName: 'ID' },
    // ...
  ],
  []
)

// AFTER
const columnDefs = useMemo<ColDef<UserResponse>[]>(
  () => [
    { field: 'id', headerName: 'ID' },
    // ...
  ],
  []
)
```

---

### 변경 2: rowSelection 속성 변경

```typescript
// BEFORE
<AgGridReact<UserResponse>
  rowSelection={{ mode: 'multiRow' }}
  // ...
/>

// AFTER
<AgGridReact<UserResponse>
  rowSelection="multiple"
  // ...
/>
```

---

## rowSelection 옵션

AG-Grid v32+ 에서 사용 가능한 값:

```typescript
rowSelection="single"      // 단일 선택 (라디오 버튼처럼)
rowSelection="multiple"    // 다중 선택 (체크박스)
```

---

## 동작 방식

### 자동 체크박스 추가

`rowSelection="multiple"` 설정 시:
1. AG-Grid가 자동으로 맨 왼쪽에 체크박스 컬럼 추가
2. 헤더에도 전체 선택 체크박스 자동 추가
3. 컬럼 정의 불필요

### 이벤트 처리 (변경 없음)

```typescript
const handleSelectionChanged = (event: SelectionChangedEvent) => {
  const selectedRows = event.api.getSelectedRows()
  setSelectedCount(selectedRows.length)
}

<AgGridReact
  onSelectionChanged={handleSelectionChanged}
/>
```

## 테스트 결과

- ✅ 체크박스 단일 표시
- ✅ 헤더 체크박스 전체 선택 작동
- ✅ 선택 개수 정상 카운트
- ✅ 일괄 삭제 버튼 정상 작동
- ✅ 콘솔 경고 메시지 제거됨
