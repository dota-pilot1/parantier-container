# 유저 관리 페이지 체크박스 기본 기능 구현

## 구현 날짜
2026-03-22

## 구현 개요
AG-Grid에 체크박스를 추가하여 다중 선택 및 일괄 작업 기능을 구현했습니다.

---

## 1. 구현된 기능

### 1.1 체크박스 선택
- ✅ 다중 선택 가능 (여러 행 체크)
- ✅ 헤더 체크박스로 전체 선택/해제
- ✅ 체크박스 컬럼 왼쪽에 고정 (pinned)
- ✅ 행 클릭으로 선택 방지 (체크박스만 클릭 가능)

### 1.2 선택 상태 추적
- ✅ 선택된 행 개수 실시간 표시
- ✅ `onSelectionChanged` 이벤트로 선택 변경 감지

### 1.3 일괄 작업 버튼
- ✅ "선택 삭제" 버튼 추가
- ✅ 선택된 항목이 없으면 버튼 비활성화
- ✅ 버튼에 선택 개수 표시

### 2.3 이벤트 핸들러 구현

#### 선택 변경 핸들러

```typescript
const handleSelectionChanged = (event: SelectionChangedEvent) => {
  const selectedRows = event.api.getSelectedRows()
  setSelectedCount(selectedRows.length)
}
```

**동작:**
1. 체크박스를 체크/해제할 때마다 자동 호출
2. 선택된 행들을 가져와서 개수를 state에 저장

#### 일괄 삭제 핸들러

```typescript
const handleBulkDelete = () => {
  const selectedRows = gridRef.current?.api.getSelectedRows()

  if (!selectedRows || selectedRows.length === 0) {
    toast.error('삭제할 사용자를 선택해주세요.')
    return
  }

  if (!confirm(`${selectedRows.length}명의 사용자를 삭제하시겠습니까?`)) {
    return
  }

  // TODO: 실제 삭제 API 호출
  toast.success(`${selectedRows.length}명의 사용자를 삭제했습니다.`)
  gridRef.current?.api.deselectAll()
}
```

**동작:**
1. 선택된 행이 없으면 에러 토스트 표시
2. 확인 대화상자 표시
3. 삭제 후 선택 해제 및 성공 토스트 표시

### 2.5 AG-Grid 설정 변경

```typescript
<AgGridReact<UserResponse>
  ref={gridRef}                                    // ref 추가
  rowData={users}
  columnDefs={columnDefs}
  defaultColDef={defaultColDef}
  pagination={true}
  paginationPageSize={20}
  paginationPageSizeSelector={[10, 20, 50, 100]}
  animateRows={true}
  rowSelection={{ mode: 'multiRow' }}              // singleRow → multiRow
  onSelectionChanged={handleSelectionChanged}      // 이벤트 핸들러 추가
  suppressRowClickSelection={true}                 // 행 클릭으로 선택 방지
  suppressCellFocus={true}
  theme={themeQuartz.withParams({
    headerHeight: 52,
    rowHeight: 60,
    fontSize: 14,
    headerFontSize: 14,
    fontFamily: '-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans", sans-serif',
  })}
/>
```

**주요 변경:**
- `ref={gridRef}`: API 접근용
- `rowSelection={{ mode: 'multiRow' }}`: 다중 선택 허용
- `onSelectionChanged={handleSelectionChanged}`: 선택 변경 감지
- `suppressRowClickSelection={true}`: 체크박스만 클릭 가능

## 3. 동작 흐름

### 체크박스 선택 시

```
1. 사용자가 체크박스 클릭
   ↓
2. AG-Grid가 내부적으로 행 선택 상태 변경
   ↓
3. onSelectionChanged 이벤트 자동 발생
   ↓
4. handleSelectionChanged 호출
   ↓
5. event.api.getSelectedRows()로 선택된 행들 가져옴
   ↓
6. setSelectedCount()로 개수 업데이트
   ↓
7. UI에 선택 개수 반영 (버튼에 표시)
```

### 일괄 삭제 버튼 클릭 시

```
1. 사용자가 "선택 삭제" 버튼 클릭
   ↓
2. handleBulkDelete 호출
   ↓
3. gridRef.current?.api.getSelectedRows()로 선택된 행 가져옴
   ↓
4. 선택된 항목 없으면 에러 토스트 표시 후 종료
   ↓
5. confirm() 대화상자 표시
   ↓
6. 사용자가 확인하면 삭제 진행
   ↓
7. TODO: 실제 API 호출 (아직 미구현)
   ↓
8. 성공 토스트 표시
   ↓
9. gridRef.current?.api.deselectAll()로 선택 해제
```
